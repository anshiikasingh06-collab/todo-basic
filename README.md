# todo-basic
A basic todo list built with vanilla JavaScript, HTML and CSS

##  Tech
Vanilla HTML, CSS, and Javascript- no libraries or frameworks.

## Features
- Add tasks
- Delete tasks
- Mark tasks as complete with strikethrough
- Persists data with localStorage
