# Todo List
A basic todo list built with vanilla HTML, CSS, and JavaScript.

## Features
- Add tasks
- Delete tasks
- Mark tasks as complete with strikethrough
- Persists data with localStorage

## Tech
Vanilla HTML, CSS, Javascript- no frameworks or libraries.
